# 你来测，我来修

> 这是给你（用户）自己动手验证用的清单。打开浏览器，挨个点过去就行。
> 任何不符合预期的地方，告诉我现象 + 怎么复现的，我来排查。

---

## 准备工作

1. 终端跑 `pnpm dev`，等看到 `Ready` 字样
2. 浏览器打开 `http://localhost:3000`
3. 第一次打开会自动建库、建欢迎页，控制台会打印 `[db] initializing ...`
4. 如果想"重头开始"：关掉 dev，删除 `data/` 文件夹，再 `pnpm dev`

---

## A. 第一次打开应该看到什么

- [ ] 左边侧边栏有且仅有一个《欢迎使用 Agent Note》👋
- [ ] 中间编辑器能看到欢迎文案
- [ ] 右边有个小目录（MiniMap），可能是空的
- [ ] 控制台没有红色错误

---

## B. 创建 / 重命名 / 删除 页面

- [ ] 侧边栏点 `+ 新建`，输入标题 `买菜清单`，回车 → 列表多一项
- [ ] 在 `买菜清单` 下面再建一个 `周三的菜`（子页面），应该缩进显示
- [ ] 点页面标题，改成 `本周买菜清单` → 侧边栏立刻同步新名字
- [ ] 把标题清空再回车 → 应该变成 `Untitled`
- [ ] 把 `周三的菜` 拖到 `买菜清单` 外面 → 变成平级
- [ ] 把一个页面拖到它自己的子页下面 → 应该被拒绝（不能形成循环）
- [ ] 删一个页面 → 它的子页、所有内容应该一起没
- [ ] 删掉当前正在编辑的页面 → 自动跳回首页，不会 404
- [ ] 刷新页面、点浏览器前进/后退 → 页面切换正常

---

## C. 编辑器基本功

- [ ] 输入几行文字 → 停一两秒 → 后台静悄悄保存了（看 Network 有 PUT 请求）
- [ ] 敲 `#` + 空格 → 变成大标题；`##` + 空格 → 二级标题
- [ ] 敲 `-` + 空格 → 无序列表；`1.` + 空格 → 有序列表
- [ ] 敲 `[]` + 空格 → 任务列表，前面有个小方框，点一下能打勾
- [ ] 敲 ``` ``` ``` + 回车 → 代码块，可以选语言（js / py / bash 等），关键字应有颜色
- [ ] 敲 `>` + 空格 → 引用样式
- [ ] 敲 `---` → 一条分割线
- [ ] 复制一段带标题、列表、代码的 Markdown 粘贴进来 → 自动转成对应块
- [ ] 拖动一个块到两个块中间 → 顺序变化，刷新后还在新位置
- [ ] 全部删空再输入 → 不报错，能保存
- [ ] Cmd/Ctrl+Z 撤销、Cmd/Ctrl+Shift+Z 重做，跨多块都能用

---

## D. 图片上传

- [ ] 把一张 PNG / JPG 拖进编辑器 → 自动上传，图片渲染出来
- [ ] 上传完保存 → 刷新页面图片还在
- [ ] 试着传一个 10MB 以上的图 → 应该被拒
- [ ] 试着传一个 `.pdf` / `.zip` / `.txt` → 应该被拒
- [ ] 在没打开任何页面的情况下传图 → 应该被拒（要 pageId）

---

## E. 反向链接 `[[ ]]`

- [ ] 在编辑器里敲 `[[买菜` → 弹出候选菜单，能看到 `买菜清单`
- [ ] 回车选中 → 文字变成蓝色可点击的链接
- [ ] Cmd/Ctrl + 点击这个链接 → 跳到 `买菜清单` 页面
- [ ] 跳过去之后点浏览器后退 → 能回到原页面
- [ ] 敲 `[[一个还没建的页面]]` + 选中「新建」 → 自动创建并跳转
- [ ] 创建完跳回原页，原来的 `[[一个还没建的页面]]` 应该自动变成了真正的链接
- [ ] 先建好 `买菜清单`，再删掉它，然后编辑器里点指向它的链接 → 弹窗问「要不要创建」，点创建可恢复

---

## F. 搜索（Cmd/Ctrl + K）

- [ ] 按 `Cmd/Ctrl + K` → 弹出搜索框
- [ ] 输入中文，比如 `买菜` → 看到标题命中 + 内容命中
- [ ] 输入英文，比如 `TODO` → 也能搜到
- [ ] 搜索结果里的关键词被 `<mark>` 高亮（黄色背景）
- [ ] 上下方向键切换、回车跳转、Esc 关闭
- [ ] 搜一个不存在的词 → 提示「没找到」而不是空白
- [ ] 连续打字 → 不要每个字都请求，应该等一下再发（看 Network 节流）

---

## G. Icon 选择

- [ ] 点页面标题旁边的图标 → 弹图标选择器
- [ ] 选个 emoji 🍎 → 标题旁和侧边栏都显示 🍎
- [ ] 切到 lucide 标签选个图标（如 `Book`）→ 也正确显示
- [ ] 刷新 → 选过的 icon 还在
- [ ] 两者都清空 → 显示默认占位符，不崩

---

## H. 大纲 / MiniMap（右侧小目录）

- [ ] 在页面里加几个标题 → 右侧 MiniMap 实时出现条目
- [ ] 点 MiniMap 的某一项 → 自动滚动到对应位置
- [ ] 滚动正文 → MiniMap 当前段落高亮
- [ ] H1 / H2 / H3 在 MiniMap 里缩进不一样
- [ ] 没有标题的页面 → MiniMap 空状态或自动隐藏

---

## I. 主题

- [ ] 右上角主题切换按钮 → 立即切换明/暗
- [ ] 刷新 → 主题保留
- [ ] 选「跟随系统」→ 改系统主题应跟随

## J. 数据完整性（进阶）

- [ ] 侧边栏建 5、6 个嵌套层级，拖来拖去 → 不出错
- [ ] 删一个含子页、孙页的页面 → 整棵子树都消失
- [ ] 控制台跑 `pnpm db:studio` → 能看到 `pages / blocks / assets / pages_fts / blocks_fts` 表

## K. 部署 / Docker（可选）

- [ ] `docker-compose up -d` → 第一次启动能正常访问
- [ ] 第一次启动时 `data/` 是空的也能自愈
- [ ] `docker-compose restart` → 不重复建表、不丢数据
- [ ] 浏览器走 nginx 端口访问 → 鉴权（htpasswd）正常
